# Extintores Report

Web app para reporte rápido de extintores de um condomínio, com:

- **Frontend estático** em GitHub Pages
- **Backend** em Google Apps Script
- **Dados** em Google Sheets
- **Fotos** em Google Drive
- **Notificações** por email
